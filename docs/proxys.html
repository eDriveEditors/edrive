VPNs

So if you haven't noticed, some laptops will have school filters enabled at home, which means that websites that are blocked at school are also blocked at home.
Here are my favorite ways of avoiding that filter entirely, at school and at home.



Psiphon is an easy to use VPN program for computers that can run exe files. If your computer can not run exe files, then go to the next option.

Pros: Easy to use, quickly connects, you don't have to change passwords every day.

Cons: Does not block Lanschool like JustFreeVPN, does not work on blocked computers.


Download Psiphon 3 ⇨


JustFreeVPN is a website-based VPN for school laptops with access to settings.

Pros: Blocks Lanschool, works on computers that can not run exes, quickly connects.

Cons: You have to go into settings and change the password once everyday.



Go to justfreevpn.com ⇨
